# Репозиторій лабораторних робіт

У цьому репозиторії створені основні папки для лабораторних робіт.

- Кореневі папки (приклад): `kr/`, `lab1/`, `lab2/`, `lab3/`, `lab4/`, `lab5/`.

Увага: у гілці `main` всередині папок лабораторних робіт всередині знаходиться лише папка `.idea` з конфігурацією IDE (файли налаштувань проєкту для JetBrains IDE). Тобто у `main` ви бачите тільки метадані IDE, а не повний вміст лабораторних робіт.

Щоб побачити конкретний вміст певної лабораторної роботи, переключіться на відповідну гілку, наприклад `lab1`, `lab2`, `lab3` або `lab4` (та інші, якщо вони є).

### Як переглянути вміст конкретної лабораторної роботи

1. Відновіть інформацію про гілки та коміти:

```bash
git fetch --all
```

2. Переключіться на потрібну гілку (наприклад `lab1`):

```bash
git checkout lab1
# або (новіша команда git)
git switch lab1
```

3. Перегляньте вміст папки лабораторної роботи:

```bash
ls -la
# або
ls -la lab1
```

Альтернативно, можна вибрати потрібну гілку через інтерфейс Git у вашому редакторі (IDE) або на GitHub, щоб побачити файли та структуру конкретної лабораторної роботи.

### Примітки

- Якщо потрібної гілки немає локально, після `git fetch --all` ви можете створити локальну гілку з віддаленої: `git checkout -b lab1 origin/lab1`.
- Файли у папці `.idea` призначені лише для конфігурації проєкту у IDE — їх не треба редагувати вручну, якщо ви не впевнені у змісті.
